# Weatherz-REACT 🌥️

This is a weather forcasting application made using **ReactJS**. This is fully real time, all the weather datas are being fetch according to the given location.

#### ⚙️ Tech Stack used in this build

- ReactJS Framwork
- Tailwind-CSS
- **API :** openweatherapi
